## 🔧 Troubleshooting

If tvbox-Swift does not start or crashes:

- Ensure your emulator has enough system resources.  
- Verify that your macOS or iOS emulator is properly set up and running smoothly.  
- Restart the app and if needed, reinstall from the release page.  
- Check your media files for compatibility issues. Certain uncommon formats may not play.  

For playback issues:

- Try converting video files into standard formats like MP4 or MOV.  
- Update your emulator software to the latest version.
